# Lumier Cosmetología Integral

Sitio web oficial de **Lumier Delicada Piel**, espacio de cosmetología, depilación láser y coaching ontológico ubicado en el corazón de Montevideo, Uruguay.

---

## Descripción

Landing page de una sola página (SPA) construida con HTML, CSS y JavaScript vanilla. Presenta los servicios, la historia de la marca, reseñas de clientes y los datos de contacto del centro.

## Servicios

### Cosmetología Holística
Técnica de acompañamiento personalizada. Incluye higiene facial, higiene de espalda, ritual de piedra gusha, aparatología, tratamientos reafirmantes, terapia de masaje en rostro, hidratación y peeling. Orientada al bienestar de la piel y el bienestar emocional.

### Depilación Láser
Tratamiento personalizado para la eliminación del vello con equipos de última generación. Se trabajan todas las áreas según las necesidades de cada persona, de forma completamente personalizada.

### Coaching Ontológico
Acompañamiento transformador que trabaja desde adentro hacia afuera: cuerpo, emoción y lenguaje. Herramienta para conectar con la autoestima, gestión emocional y propósito personal.
